# React Tree

Semantic grid view for tree structures.
Inspired by [Backgrid](https://github.com/wyuenho/backgrid)
and [Marionette Tree](https://github.com/juanca/marionette-tree).

## Getting started

    git clone git@github.com:juanca/react-tree.git
    npm install

### Compiling library

    npm run compile

### Developing locally

    npm start

### Testing locally

    npm test

## API

Out of the box:

- Collection (of data)
- Columns
